Servidor REST utilizando API REST-RS com CDI e JPA

A partir de tal endereço, cada classe anotada com @Path irá definir caminhos adicionais. Por exemplo, para acessar a API de acesso a dados de usuários, temos algumas URIs como: Consultar voos por origem, destino, data de partida e faixa de preço Cadastrar reserva do voo Alterar reserva do voo Excluir reserva do voo

  • Consultar voos por origem e destino
  • Consultar voos por origem, destino e data de partida
  • http://localhost:8080/api/usuario/1 para buscar usuário pelo ID
  • http://localhost:8080/api/usuario/cpf/11111111111 para buscar usuário pelo CPF
